# 📌 About the project
This is a simple responsive front-end project that recreates the Spotify search page. 
Contains a feature that allows you to search for artists using a search bar.

# ⚙️ How to execute the project

## 1. Clone the repository:

```
git clone https://github.com/pplace11/Spotify-Imersao.git
```
```
cd Spotify-Imersao
```

## 3. Install the Live Server extension in Visual Studio Code:

- *Abra o Visual Studio Code.*
- *Vá até a aba de extensões (ícone de quadrado no lado esquerdo ou use o atalho Ctrl+Shift+X).*
- *Procure por "Live Server" e clique em "Install" na extensão desenvolvida por Ritwick Dey.*
  
## 5. Install Node.js dependencies:

```
npm install
```

## 7. Configure json-server:

```
npm install -g json-server
```

## 9. Start json-server:

```
json-server --watch api-artists/artist.json --port 3000
```

## 🗂️ Project Structure

```
📁 Spotify-Imersao/
├── 📂 src/         # JavaScript scripts and application logic
│   ├──📂 assets/         # Static resources like CSS, JS and images
│   ├──📂 styles/         # Contains CSS files that define the application's styles
│   │   ├── 📄 sidebar-footer.css      # Application styles
│   │   ├── 📄 reset.css               # Reset styles for cross-browser consistency
│   │   ├── 📄 main-content.css        # Defines styles for the application's main content area
│   │   ├── 📄 vars.css                # Defines CSS variables for fonts and colors
│   │   ├── 📄 media-queries.css       # Sets responsive styles for different screen sizes
├── 📂 api-artists/  # Contains artist data
│   ├── 📄 artist.json      # JSON file with artist data
├── 📄 script.js     # JavaScript script for manipulating artist data
├── 📄 index.html    # Application main page
└── 📄 README.md     # Project documentation
```
